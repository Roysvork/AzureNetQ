Easy Azure

AzureNetQ is a client API for Microsoft Azure Service Bus & Service Bus for Windows on .NET. You can find it as a NuGet package here.

Due to similarities in the API some information and documentation from EasyNetQ may still apply.

Easy to use

We designed EasyNetQ to be as simple as possible. Get started publishing and subscribing with RabbitMQ with just a few lines of code:

To connect to Service Bus...


<add key="Microsoft.ServiceBus.ConnectionString" value="Endpoint=sb://servicebus/ServiceBusDefaultNamespace;StsEndpoint=https://servicebus:10355/ServiceBusDefaultNamespace;RuntimePort=10354;ManagementPort=10355" />
          

var bus = AzureBusFactory.CreateBus();

To publish...


bus.Publish(message);

To subscribe...


bus.Subscribe<MyMessage>(msg => 
      Console.WriteLine(msg.Text), 
      x => x.WithSubscription("my_subscription"));

Easy to install

AzureNetQ is on NuGet. To install it, run the following command in the Package Manager Console


  PM> Install-Package AzureNetQ
